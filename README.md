# banner-lhsl-project

## 本地端部署
#### step1
```
Clone專案至本地端
```
#### step2
```
刪除 package-lock.json 這份檔案
```
#### step3
```
終端機輸入
npm install
```
#### step4
```
終端機輸入
npm run serve
```

## 遠端部署
#### step1
```
Clone專案至本地端
```
#### step2
```
將/dist 資料夾整包放入serve的對外空間
```
#### step3
```
在瀏覽器中輸入： {server ip}/{dist資料夾在serve中的路徑}/{foxconn}/index.html
若無意外，即可訪問頁面
```
