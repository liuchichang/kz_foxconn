# banner-lhsl-project

## Project setup
#### 本地端測試 step1
刪除 package-lock.json 這份檔案，然後：
```
npm install
```

### Compiles and hot-reloads for development
#### 本地端測試 step2
```
npm run serve
```

### Compiles and minifies for production
#### 本地端測試不需要
```
npm run build
```

### Lints and fixes files
#### 本地端測試不需要
```
npm run lint
```
