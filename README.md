# foxconn 亞灣新創環控系統

## 本地端部署
#### step 1
```
Clone專案至本地端
```
#### step 2
```
終端機輸入(node環境下)
npm install
```
#### step 3
```
終端機輸入
npm run serve
```

## 遠端部署
#### step 1
``` 
Clone專案至本地端
```
#### step 2
```
將/dist 資料夾整包放入serve的對外空間
```
#### step 3
```
在瀏覽器中輸入： {server ip}/{dist資料夾在serve中的路徑}/{foxconn}/index.html
若無意外，即可訪問頁面
```
